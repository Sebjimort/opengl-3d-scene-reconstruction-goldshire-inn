## English Version

This repository contains a technical 3D reconstruction of the iconic "Goldshire Inn" from World of Warcraft. Developed as a university project for the Computer Engineering program, this application demonstrates the practical application of computer graphics concepts, rendering the environment using custom shaders and asset extraction techniques.

### Technologies Used
* **C++**
* **OpenGL**
* **Autodesk Maya** (for 3D environment modeling and management)
* **wow.export** (for raw asset extraction and pipeline integration)

Comprehensive documentation has been created for both developers and end-users. Please refer to the following manuals for detailed architecture, implementation specifics, and usage instructions:

## Versión en Español

Este repositorio contiene una reconstrucción técnica en 3D de la icónica posada "Goldshire Inn" de World of Warcraft. Desarrollado como un proyecto universitario para la carrera de Ingeniería en Computación, esta aplicación demuestra la aplicación práctica de conceptos de computación gráfica, renderizando el entorno mediante el uso de shaders personalizados y técnicas de extracción de assets.

### Tecnologías Utilizadas
* **C++**
* **OpenGL**
* **Autodesk Maya** (para el modelado y gestión del entorno 3D)
* **wow.export** (para la extracción de assets en bruto y flujo de trabajo)
